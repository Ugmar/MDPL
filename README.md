# Машинно-зависимые языки программирования (MDPL)

Репозиторий содержит комплекс лабораторных работ, посвященных изучению архитектуры процессоров семейства x86, низкоуровневого программирования на языке ассемблера (NASM/MASM/TASM), интеграции ассемблерного кода с языками высокого уровня (C, C++), использованию векторных расширений (FPU, SSE), а также разработке системных приложений.

## 🛠 Технологический стек

* **Языки программирования:** Assembly (x86-32 / x86-64), C, C++ (стандарты C99 / C++17)
* **Ассемблеры и компиляторы:** NASM, MASM, GCC, G++
* **Система сборки:** GNU Make (`makefile`)
* **Окружение и GUI:** Linux (Ubuntu/WSL2), Qt / C++ (для графических интерфейсов лабораторных работ)

## 📂 Структура репозитория и описание лабораторных работ

### 🔹 Базовый курс Ассемблера (Лабораторные 1–4)
Изучение основ синтаксиса, структуры исполняемых файлов, сегментации памяти и взаимодействия модулей.
* **1_LAB (`hello.asm`):** Первая программа. Вывод строки в стандартный поток вывода, базовая инициализация сегментов.
* **2_LAB (`main.asm`):** Изучение режимов адресации, работы с регистрами общего назначения и простейших арифметических команд.
* **3_LAB (Модульные программы):** * Разделение кода на несколько модулей (`module1.asm`, `module2.asm`).
  * Изучение директив `PUBLIC` и `EXTERN` для организации межмодульных вызовов.
  * Реализация ветвления и циклических конструкций в ассемблерных файлах (`lr03_1` — `lr03_4`).
* **4_LAB (Обработка матриц):**
  * `matrix.asm`, `utils.asm`.
  * Работа с двумерными массивами (матрицами) на низком уровне. Ввод/вывод элементов, адресация со смещением, базовые алгоритмы трансформации матриц.

### 🔹 Взаимодействие со строками и прерывания (Лабораторные 5–8)
* **5_LAB / 6_LAB:** Обработка текстовых данных и строк. Работа со строковыми инструкциями (`MOVS`, `CMPS`, `SCAS`).
* **7_LAB:** Реализация пользовательских обработчиков прерываний, работа с системным таймером или клавиатурным прерыванием.
* **8_LAB:** Организация функций ввода-вывода и преобразования типов (например, ассемблерная процедура `input_digit` с самостоятельной очисткой стека).

### 🔹 Смешанное программирование и высокопроизводительные вычисления (Лабораторные 9–12)
* **9_LAB (Математический сопроцессор FPU):**
  * Реализация вычислений с плавающей запятой.
  * Написание эффективных ассемблерных вставок для вычисления сложных тригонометрических и степенных функций (например, $y = \sin(x^2 + 5x)$).
  * Исследование точности вычислений FPU по сравнению со стандартными типами данных в C.
* **10_lab (Интеграция C++ и Ассемблера):**
  * Скрипты `task1.cpp`, `task2.cpp`, `task3.cpp`.
  * Передача параметров через стек в соответствии с соглашениями о вызовах (`cdecl`, `stdcall`).
  * Реализация итерационных численных алгоритмов (например, метод деления пополам или метод Ньютона для поиска корней на заданном отрезке).
* **11_lab (Графический интерфейс на Qt):**
  * Модули `MainWindow.cpp`, `GridWidget.cpp`.
  * Интеграция вычислительного ядра (на C/Assembly) с графическим интерфейсом на базе библиотеки Qt.
  * Визуализация результатов вычислений и расчетных сеток в реальном времени.
* **12_lab (Системное программирование на C):**
  * Файлы `task1.c`, `task2.c`.
  * Прямая работа с системными вызовами ОС, оптимизация строковых операций с использованием векторных инструкций процессора (SSE/AVX).
